# MeshMaker 
## 简介
	MeshMaker是在Unity Runtime下，编辑网格的项目，可结合XObject项目，实现导入导出。
	因为MeshMaker实际为工作项目中一个功能模块，移出来分享，代码并非完全针对MeshMaker。
